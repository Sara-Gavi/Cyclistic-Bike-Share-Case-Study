## Data Location

The data is located on a public website called **Divvy Tripdata**, hosted on **Amazon Web Services (AWS)**. From this website, the monthly bike trip files can be downloaded in ZIP format. No registration is required to access the data.

Official source:  
https://divvy-tripdata.s3.amazonaws.com/index.html

---

## ROCCC Evaluation

The data is **reliable** because it comes directly from the company that manages the bike-sharing system and is hosted on AWS.  
The data is **original** because it has not been modified by third parties.  
The data is **complete enough** to analyze user behavior, ride duration, and bike usage.  
The data is **current**, as it belongs to the years 2023 and 2024.  
The data is **cited**, as it comes from an official public source.

---

## Data Limitations

The dataset does not contain personal user information for privacy reasons. This means it is not possible to track individual users over time or analyze personal behavior patterns. The analysis can only be done at an aggregated level (members vs. casual riders).

---

## Licensing, Privacy, Security and Accessibility

The data is public and can be downloaded for free from Divvy Tripdata (AWS). The dataset does not contain personal user information, so privacy is respected. Since it is hosted on AWS, the data is considered secure and easily accessible.

---

## Data Integrity Check

It was verified that the files come directly from the official Divvy Tripdata source and have not been modified.

---

## Data Usefulness for the Analysis

The dataset allows us to analyze how annual members and casual riders use the bikes using variables such as ride time, date, time of day, and user type.

---

## Data Issues

So far, no serious problems have been detected. Some columns are still under review, which is normal at this stage of the project.

Los datos están ubicados en un sitio web público llamado **Divvy Tripdata**, alojado en **Amazon Web Services (AWS)**. Desde allí se pueden descargar los archivos mensuales de viajes en bicicleta en formato ZIP. No se requiere registro para acceder a ellos.

## Evaluación ROCCC

Los datos utilizados son **confiables** porque provienen directamente de la empresa que gestiona el sistema de bicicletas y están alojados en AWS.

Son **suficientemente completos**, puesto que contienen las variables necesarias para analizar el comportamiento de los usuarios.

## Limitaciones de los datos

Esto significa que no es posible identificar a un mismo usuario a lo largo del tiempo ni analizar patrones individuales de comportamiento. El análisis solo puede realizarse a nivel agregado (miembros vs. usuarios ocasionales).
